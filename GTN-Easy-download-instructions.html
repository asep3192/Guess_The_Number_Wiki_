Guess The Number (GTN) Easy gamemode download Instructions

Step by Step Instructions

Note: GTN (Guess The Number) is only supported on Windows

    1: Go to the Guess The Number easy gamemode Page. You can do this by clicking on the the image in the top corner with a "1" (one) and a "?" (question mark) in the center of the image.

      Step 2. Depending on your settings the games download file will either just download to your default downloads locations or ask you where you want to download the zip file.
      Step 3. Click on either the itch.io download link or the Website download link (V1.2.1)
      Note: that the itch io download link is always online, while the Website link will go offline from time to time.
      Step 4. Go to the location that the zip file was downloaded to and extract it (You can do this by right clicking on the zipped folder and clicking Extract all. Then click Extract after choosing your extract location.)
      Step 5: Open the extracted folder run the .exe of Guess The Number 5.1. Windows smart screen is going to alert you that windows has protected your PC. This is normal for running Guess The Number (GTN) for the first time. When it appears click on 'More Info' and then 'Run Anyway'.
