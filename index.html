Welcome to the Guess The Number Wiki

Guess The Number is a game created by Asep1582 (asep3192)

Where you guess a number between 0 and the number limit set with a certain amount of Gueses. There is 1 gamemodes currently released and one gamemode in selected beta. These are: GTN - Medium Easy and GTN Medium.

There are multiple game modes of Guess The Number. (Gamemodes Listed below (in order))

  • Guess The Number Easy Gamemode (Instructions to download here)
  • Guess The Number Medium Easy GameMode
  • Guess The Number Medium Gamemode
  • Guess The Number Hard Mode
  • Guess The Number Extreme Hard Mode (discontinued)
  • Guess The Number Extra Hard Mode (The Hardest Gamemode)

Where can I find Guess The Number game (.exe) ?

You can find Guess The Number on this website, itch.io, and the creators Github Account: Asep1582/Asep3192
