This is Medium Gamemode

Description: Medium Gamemode is the third gamemode in the Guess The Number Gamemodes and was originally the second gamemode in line. But was moved to 3rd when Medium Easy Gamemode was created. In this Gamemode you have 8 guesses and have to guess a randomly generated number between 0 and 300.

Where can find this Gamemode?

You can find Medium Gamemode at this link: Coming Soon!

Or on the Creators Github. Which you can either go to the direct link to the game or by clicking on the Github Image in the top corner, which brings you to the creators Github Profile.
